# Project: [项目名称]

## Description
这是一个基于通过AI智能体+浏览器操作构建的智能化小红书运营系统

## Commands (常用命令)
* **Run App**: `python main.py` / `npm run dev`
* **Run Tests**: `pytest tests/` / `npm test`
* **Lint/Format**: `ruff check .` / `eslint .`
* **Install Deps**: `pip install -r requirements.txt`

## Code Style Guidelines (代码规范)
### General
* **Language**: 使用 Python 3.11
* **Formatting**: 严格遵守 PEP 8 (Python) 或 Prettier (JS/TS) 配置。

## 代码构建原则
* 遵循奥卡姆剃刀原则，在充分满足需求的前提下，保持结构简洁
* 良好的代码文件结构，避免单一代码过于冗长
* 良好的注释信息
* 需要编写规范清晰的中文注释
* 如果没有明确要求，不要构建描述性md文本，你的主要工作以构建代码为主
* 如果有需要引入的模块或代码，将import代码放在文件开头位置
