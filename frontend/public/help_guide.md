# AIMediaOps 使用指南

## 产品简介

AIMediaOps 是一个智能化的小红书运营管理平台，支持自动化管理小红书账号、定时发布内容，提升运营效率。

## 核心功能

### 1. 任务管理

#### 1.1 创建任务

1. 点击左侧面板的"新建任务"按钮
2. 填写任务信息：
   - **账户ID**：小红书账号ID（必填）
   - **账户名称**：小红书账户名称（必填）
   - **账号运营要求**：描述账号运营的目标和要求
   - **账号内容主题**：指定内容主题（如：科技、美食、旅游等）
   - **内容风格**：指定内容风格（如：专业、可爱、幽默等）
   - **目标受众**：指定目标受众群体
   - **执行间隔**：任务执行间隔时间（15分钟-3小时）
   - **有效时间范围**：任务执行的时间范围（可选，支持"无限制"）
   - **任务结束时间**：任务的结束日期

3. 点击"创建"按钮完成任务创建

#### 1.2 编辑任务

1. 在任务列表中，点击任务卡片右上角的"编辑"图标
2. 在弹窗中修改任务属性：
   - 可以修改账号运营要求、内容主题、风格、目标受众等
   - 可以调整执行间隔和时间范围
   - 可以更新任务结束时间
3. 点击"保存"按钮应用更改

#### 1.3 暂停/恢复任务

- **暂停任务**：点击任务卡片上的"暂停"按钮，任务将暂停执行（正在运行的任务会等待当前执行完成后暂停）
- **恢复任务**：点击任务卡片上的"恢复"按钮，任务将恢复执行

#### 1.4 立即执行

点击任务卡片上的"立即执行"按钮，可以立即执行该任务一次，不受时间范围限制。

#### 1.5 删除任务

点击任务卡片右上角的"删除"图标，可以删除任务。

### 2. 调度器管理

#### 2.1 启动调度器

点击页面上的"启动调度器"按钮，调度器将开始运行，按照任务配置自动调度执行任务。

#### 2.2 停止调度器

点击页面上的"停止调度器"按钮，调度器将停止运行，所有任务将暂停执行。

#### 2.3 刷新状态

点击"刷新状态"按钮，可以更新调度器和任务的最新状态。

### 3. 任务设置

在左侧设置面板中，可以对当前选中的任务进行快速设置：

- **执行间隔**：调整任务的执行间隔（15分钟-3小时，以15分钟为刻度）
- **有效时间范围**：
  - 可以选择"无限制"，任务将在任何时间执行
  - 或者设置具体的开始和结束小时（0-23点）
- **结束时间**：设置任务的结束日期

### 4. 账号登录

对于未登录的账号，系统会提示进行登录：

1. 点击任务卡片上的"登录"按钮
2. 系统将生成登录二维码
3. 使用小红书App扫描二维码
4. 扫码后点击"确认"按钮
5. 系统将验证登录状态，登录成功后即可正常使用

### 5. 日志查看

1. 点击任务卡片上的"查看日志"按钮
2. 在弹窗中查看任务的实时执行日志
3. 可以按日志级别过滤（全部、INFO、WARNING、ERROR）
4. 日志会自动刷新，显示最新的执行记录

### 6. 资源管理

#### 6.1 查看图片

1. 点击任务卡片上的"查看资源"按钮
2. 切换到"图片"标签页
3. 可以查看任务生成的所有图片
4. 图片以缩略图形式展示，点击可查看大图

#### 6.2 知识库管理

1. 切换到"知识库"标签页
2. 可以查看和编辑 `text.md` 知识库文件
3. 支持导入、导出和在线编辑
4. 知识库文件用于指导AI生成内容的风格和主题

### 7. 任务列表筛选

- **状态筛选**：使用"全部状态"下拉菜单，可以按任务状态筛选（全部、等待中、运行中、已暂停、已完成、错误）
- **账户筛选**：使用"全部账户"下拉菜单，可以按账户筛选任务
- **刷新**：点击"刷新"按钮，更新任务列表

## 常见问题

### Q1: 任务为什么没有执行？

**可能的原因：**
1. 调度器未启动 - 检查页面上的调度器状态，点击"启动调度器"
2. 任务已暂停 - 检查任务状态，点击"恢复"按钮
3. 不在执行时间范围内 - 检查任务的有效时间范围设置
4. 任务已到期 - 检查任务的结束时间

### Q2: 如何设置任务在任何时间都可以执行？

在任务设置中，将"有效时间范围"设置为"无限制"即可。

### Q3: 任务执行间隔可以设置多久？

执行间隔范围为 15分钟 到 3小时，以15分钟为刻度单位。

### Q4: 如何查看任务的执行历史？

点击任务卡片上的"查看日志"按钮，可以查看任务的详细执行日志。

### Q5: 账号登录失败怎么办？

1. 确保二维码未过期（有效期180秒）
2. 使用小红书App扫码，不要使用其他工具
3. 扫码后及时点击"确认"按钮
4. 如果仍然失败，可以重新点击"登录"生成新的二维码

### Q6: 如何修改任务的执行时间？

有两种方式：
1. **快速设置**：选中任务后，在左侧设置面板中调整执行间隔和时间范围
2. **详细编辑**：点击任务卡片上的"编辑"按钮，在弹窗中修改所有属性

### Q7: 任务暂停和停止的区别？

- **暂停**：任务暂停执行，但保留在任务列表中，可以随时恢复
- **删除**：任务被永久删除，无法恢复

### Q8: 立即执行和定时执行的区别？

- **立即执行**：不受时间范围限制，立即执行一次
- **定时执行**：按照配置的执行间隔和时间范围，由调度器自动执行

## 最佳实践

1. **合理设置执行间隔**：建议根据内容类型和账号定位，设置合适的执行间隔（如：每天1-3次）
2. **设置有效时间范围**：建议在工作时间内执行任务，避免深夜执行
3. **定期查看日志**：定期检查任务执行日志，及时发现问题
4. **维护知识库**：定期更新知识库文件，确保AI生成的内容符合预期
5. **账号安全**：妥善保管账号信息，避免泄露
