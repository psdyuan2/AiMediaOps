# Web 测试页面使用说明

## 访问方式

启动 API 服务后，访问根路径即可看到 Web 测试页面：

```
http://localhost:8000/
```

## 功能说明

### 1. 调度器控制区

- **启动调度器**: 开始执行任务调度
- **停止调度器**: 停止调度器（等待当前任务完成）
- **刷新状态**: 手动刷新调度器状态
- **状态显示**: 实时显示调度器运行状态和任务统计

### 2. 任务列表区

- **任务表格**: 显示所有任务的关键信息
  - 任务ID（前8位）
  - 账户名称
  - 任务状态（带颜色标识）
  - 下次执行时间
  - 执行轮次
  - 操作按钮

- **筛选功能**:
  - 按状态筛选（全部/等待执行/正在执行/已暂停/已完成/错误）
  - 按账户筛选

- **操作功能**:
  - **暂停**: 暂停任务执行
  - **恢复**: 恢复已暂停的任务
  - **详情**: 查看任务详细信息
  - **删除**: 删除任务（需要确认）

### 3. 创建任务区

- **必填字段**:
  - 账户ID
  - 账户名称

- **可选字段**:
  - 查询内容
  - 帖子主题
  - 内容风格
  - 目标受众
  - 执行间隔（默认3600秒）
  - 有效时间范围（默认8-22时）
  - 任务结束时间（默认30天后）

## 状态颜色说明

- **灰色**: 等待执行 (pending)
- **蓝色**: 正在执行 (running)
- **黄色**: 已暂停 (paused)
- **绿色**: 已完成 (completed)
- **红色**: 错误 (error)

## 自动刷新

- 调度器状态：每5秒自动刷新
- 任务列表：每10秒自动刷新
- 也可以手动点击"刷新"按钮

## 注意事项

1. **删除任务**: 删除操作需要确认，不可恢复
2. **停止调度器**: 会等待当前运行的任务完成后再停止
3. **创建任务**: 如果账户ID已存在任务，会提示错误
4. **浏览器兼容**: 建议使用 Chrome、Firefox 或 Safari 最新版本

## 故障排查

### 页面无法加载

1. 检查 API 服务是否启动
2. 检查端口是否正确（默认8000）
3. 查看浏览器控制台错误信息

### API 调用失败

1. 检查网络连接
2. 查看浏览器控制台的错误信息
3. 检查 API 服务日志

### 数据不更新

1. 点击"刷新"按钮手动刷新
2. 检查浏览器控制台是否有错误
3. 检查 API 服务是否正常运行
